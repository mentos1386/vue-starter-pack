# Vue-Starter-Pack

> vue-resource
> vue-router
> bootstrap 4
> font-awesome

## Config

In file `./config/config.js`

```js
module.exports = {
  API_ROOT: 'https://localhost',
  OTHER_ENV: 'WHATEVER VALUE'
}
```

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
